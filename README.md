# Spammer

## Overview
A discord.py discord bot called Spammer.
With this bot, you can;

**NUKE SERVERS**
**DELETE ALL THE CHANNELS**
**CREATE RANDOM CHANNELS**
**DELETE EVERYTHING**
**CREATE SOME RANDOM THINGS**

## Using on the server

**Type !help for the comments.**
**!gg for nuking, !gg noban with nuking but not banning.**
**!party for randomness, !party noban with randomness but not banning.**

## Installation
```bash
git clone https://github.com/muhabbetkusudayi/spammer.git
cd spammer
```

